○セットアップ

　本体へ、以下の順でセットアップします。

　※Windowsの場合の説明となります。
　　Macの場合、エクスプローラーをFinderと、読み替えてください。

　※以下は、プロローグの場合のセットアップ手順となります。
　　前編・後編・エピローグをセットアップする場合は、
　　ファイル末尾（_p）を_1･_2･_eに置き換えてください。
　　（例　前編のプログラムファイルであれば、port_p.hex → port_1.hex）

　《実機で、PC-Gリンク等の転送ソフトをお使いの場合》

　　本体のリセットボタンを押してください。
　　（本体のメモリを全てクリアすれば、リセットしなくても大丈夫です）

　　テキストエディタの「Sio」・「Format」を選択し、
　　baud rateを「9600」で設定してください。

　　BASICモードからMONで機械語モニタを起動します。
　　※USERによる、機械語のユーザーエリアの確保はしないでください。

　　Rと入力してEnterキーを押し、プログラムファイル（port_p.hex）を転送します。

　　Rと入力してEnterキーを押し、データファイル（data_p.hex）を転送します。

　《SHARP PC-G850/G815/E200エミュレータ g800をお使いの場合》

　　（実機ROMファイルがある場合）

　　　g800を起動します。

　　　リセットボタンを押してください。
　　　（本体のメモリを全てクリアすれば、リセットしなくても大丈夫です）

　　　テキストエディタの「Sio」・「Format」を選択し、
　　　baud rateを「9600」で設定してください。

　　　BASICモードからMONで機械語モニタを起動します。
　　　※USERによる、機械語のユーザーエリアの確保はしないでください。

　　　エクスプローラーからg800のウインドウへ、プログラムファイル（port_p.h
　　　ex）をドラッグ&ドロップすると、g800へ転送するファイルがセットされます。

　　　Rと入力してEnterキーを押すと、ドロップしたファイルが転送されます。

　　　エクスプローラーからg800のウインドウへ、データファイル（data_p.hex）をド
　　　ラッグ&ドロップすると、g800へ転送するファイルがセットされます。

　　　Rと入力してEnterキーを押すと、ドロップしたファイルが転送されます。

　　（実機ROMファイルが無い場合）

　　　g800を起動します。

　　　BASICモードからMONで機械語モニタを起動します。
　　　※USERによる、機械語のユーザーエリアの確保はしないでください。

　　　エクスプローラーからg800のウインドウへ、プログラムファイル（port_p.h
　　　ex）をドラッグ&ドロップすると、ファイルが転送されます。

　　　エクスプローラーからg800のウインドウへ、データファイル（data_p.hex）をド
　　　ラッグ&ドロップすると、ファイルが転送されます。

○ポートアイランド連鎖殺人事件　−ＳＴＯＲＹ−

　突然、部下のヤスが、血相を変えて飛び込んできた。
　「大変です、ボス！　ローンヤマキンの社長・山川耕造が自殺しました！」
　「な、なにーっ！？」
　俺は、捜査一課の腕利き刑事である。
　山川耕造・・・サラ金会社・社長。その悪質な経営振りで、人を自殺に追い込んだこ
　とはあったが、まさか、その耕造が自殺してしまうとは・・・。
　俺には、どうにも信じられなかった。
　「しかし、ボス。耕造は完全な密室で死んでいたんですよ」
　「話してみてくれ」と、俺。
　「ええ、見つけたのは、耕造の秘書の沢木文江って女性なんですが、耕造がいつまで
　　たっても会社に現れないので、耕造の屋敷まで様子を見にいき、そして」
　ヤスの話をまとめてみると、こうだった。
　屋敷に着いた文江は、守衛の小宮という老人に中に入れてもらう。
　「たぶん書斎におるんじゃろう」と、小宮老人。
　しかし、書斎のドアには鍵がかかっていて、いくら呼んでも返事がない。おかしいと
　思った２人は、体当たりしてドアを叩き開け、そこで耕造の死体を発見したというこ
　とだった。
　「しかも、ボス。小宮が気づいたんですが、書斎のドアには内側から鍵が差し込まれ
　　ていたそうです」
　ヤスの話を聞きながら、俺は煙草を灰にする作業に専念していた。
　これは、本当に自殺なのだろうか・・・？

○操作方法

　《タイトル画面》

　　Sキー 　　　音のOFF/ONの切り替え
　　　　　　　　※本体に圧電ブザーが取り付けられていない場合、
　　　　　　　　　ONにしても音は鳴りません。
　　SPACEキー 　ゲームスタート

　《ゲーム》

　　入力プロンプト(ﾎﾞｽ?)が表示されたら、
　　名詞＋動詞の形式で入力してEnterを押すことにより、
　　ヤスに命令します。
　　入力はローマ字となります。
　　（ポケコンPC-G850のローマ字入力方法に準拠しています）
　　名詞と動詞の間は、空白を１文字入れてください。

　　（入力例）
　　　KIKIKOMI SHIRO → ｷｷｺﾐ ｼﾛ
　　　HANAKUMACHOU IKE → ﾊﾅｸﾏﾁｮｳ ｲｹ

　　持っているは、「ﾓﾁﾓﾉ」で確認できます。

　　以下のコマンドにより、音のON/OFFの切り替えができます。
　　ｻｳﾝﾄﾞ ｵﾝ　音を出します。
　　ｻｳﾝﾄﾞ ｵﾌ　音を消します。

○実際の捜査方法

　では、実際にどういうふうにゲームが進んでいるか、やってみましょう。
　捜査の手始めは、何といっても現場検証です。
　移動したい場所へ行くよう、ヤスに命令すると、指定した場所へ移動します。
　そこで、「ﾔﾏｶﾜﾉﾔｼｷ ｲｹ」と入力してEnterを押すと、山川の屋敷へ移動します。
　その後、「ﾔｼｷﾉﾅｶ ｲｹ」と命令し、屋敷の中に入ります。
　書斎では、「ｼﾀｲ ｼﾗﾍﾞﾛ」「ｼｲﾝ ｼﾗﾍﾞﾛ」「ｷｮｳｷ ｼﾗﾍﾞﾛ」で、死体や凶器を調べ、
　その他、部屋の中も隅々まで調べてみましょう。
　もし、取れるものがあれば、証拠品として取っておきます。
　念のため、「ｺｳｿﾞｳ ｼｮｸ」で、耕造の身の上も知っておきます。
　また、現場付近での聞き込みも大切です。
　「ﾊﾅｸﾏﾁｮｳ ｲｹ」で花隈町に行き、ヤスに「ｷｷｺﾐ ｼﾛ」と命令。聞き込みをしてください。
　ある人物が浮かび上がってくるかもしれません。
　そこで、その人物の職業を調べます。
　さらに、人の尋問も重要です。
　「ｿｳｻﾎﾝﾌﾞ ｲｹ」で捜査本部へ戻り、取り調べ室に「ﾌﾐｴ ﾖﾍﾞ」で、第一発見者の文江を
　呼びます。
　「ﾌﾐｴ ｼｮｸ」で彼女のことを調べた後、「ｺｳｿﾞｳ ｷｹ」や「ｱﾘﾊﾞｲ ｷｹ」で、耕造のことや、
　事件の夜のアリバイを聞いてみましょう。

　・・と、これは、ほんの一例です。
　どこに行くか。誰から調べるか、それは、ボスであるあなたの自由です。
　何か手掛かりを掴んだら、文江のことなど放っておき、その手掛かりからどんどん究明
　していっても構いません。
　さあ、何から始めますか、ボス？

○ゲームのヒント

　Ｑ．捜査本部での、登場人物に対する尋問の仕方がよく判りません。
　Ａ．「ｱﾘﾊﾞｲ ｷｹ」でアリバイを聞いたり、
　　　耕造やその他登場人物のことを聞いたり「◯◯（登場人物） ｷｹ」、
　　　持っているものを、「◯◯（持ち物） ﾐｾﾛ」のように、見せてみてください。
　　　また、「ｼｬｼﾝ ﾄﾚ」で、写真を撮っておいた方がいいでしょう。
　　　後々、役に立つかもしれません。

　Ｑ．捜査現場での進め方がよく判りません。
　Ａ．「ｷｷｺﾐ ｼﾛ」と、こまめに聞き込みをしてください。
　　　また、「ｼｬｼﾝ ﾐｾﾛ」で、撮影した写真を見せると、目撃者がいるかもしれません。
　　　他、画面をよく見て、何か気になる箇所があれば、「◯◯（対象物） ｼﾗﾍﾞﾛ」と
　　　確認すると、新たな発見があるかもしれません。
　　　もし、重要参考人がいれば、「◯◯（登場人物） ｻｶﾞｾ」で、くまなく探すことも
　　　大切です。

　Ｑ．なかなか話が進展しません。
　Ａ．登場人物や浮上した人物の職業を「◯◯（登場人物） ｼｮｸ」で調べたり、
　　　独自にアリバイを調査するため、「◯◯（登場人物） ｱﾘﾊﾞｲ」で調べてみてください。
　　　もし、新たな登場人物が浮かび上がったら、捜査本部で「◯◯（登場人物） ﾖﾍﾞ」
　　　で呼び、尋問をすることを忘れずに。
　　　また、捜査本部から、新たな報告があるかもしれません。
　　　定期的に、「ｿｳｻﾎﾝﾌﾞ ｲｹ」で、捜査本部へ足を運んでみてください。

　Ｑ．俊之のメモの暗号が解けません。
　Ａ．「こめいちご」の暗号を解いてみると、解答の糸口が見えてくるかと思います。
