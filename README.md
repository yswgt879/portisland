## セットアップ

本体へ、以下の順でセットアップします。

※Windowsの場合の説明となります。<br>
　Macの場合、エクスプローラーをFinderと、読み替えてください。

※以下は、プロローグの場合のセットアップ手順となります。<br>
　前編・後編・エピローグをセットアップする場合は、<br>
　ファイル末尾（_p）を`_1`･`_2`･`_e`に置き換えてください。<br>
　（例　前編のプログラムファイルであれば、`port_p.hex` → `port_1.hex`）

### 実機で、PC-Gリンク等の転送ソフトをお使いの場合

* 本体のリセットボタンを押してください。<br>
（本体のメモリを全てクリアすれば、リセットしなくても大丈夫です）
* テキストエディタの「Sio」・「Format」を選択し、baud rateを「`9600`」で設定してください。
* BASICモードから`MON`で機械語モニタを起動します。<br>
※USERによる、機械語のユーザーエリアの確保はしないでください。
* `R`と入力してEnterキーを押し、プログラムファイル（`port_p.hex`）を転送します。
* `R`と入力してEnterキーを押し、データファイル（`data_p.hex`）を転送します。

### SHARP PC-G850/G815/E200エミュレータ g800をお使いの場合

#### 実機ROMファイルがある場合

* g800を起動します。
* リセットボタンを押してください。<br>
（本体のメモリを全てクリアすれば、リセットしなくても大丈夫です）
* テキストエディタの「Sio」・「Format」を選択し、baud rateを「`9600`」で設定してください。
* BASICモードから`MON`で機械語モニタを起動します。<br>
※USERによる、機械語のユーザーエリアの確保はしないでください。
* エクスプローラーからg800のウインドウへ、プログラムファイル（`port_p.hex`）をドラッグ&ドロップすると、g800へ転送するファイルがセットされます。
* `R`と入力してEnterキーを押すと、ドロップしたファイルが転送されます。
* エクスプローラーからg800のウインドウへ、データファイル（`data_p.hex`）をドラッグ&ドロップすると、g800へ転送するファイルがセットされます。
* `R`と入力してEnterキーを押すと、ドロップしたファイルが転送されます。

#### 実機ROMファイルが無い場合

* g800を起動します。
* BASICモードから`MON`で機械語モニタを起動します。<br>
※USERによる、機械語のユーザーエリアの確保はしないでください。
* エクスプローラーからg800のウインドウへ、プログラムファイル（`port_p.hex`）をドラッグ&ドロップすると、ファイルが転送されます。
* エクスプローラーからg800のウインドウへ、データファイル（`data_p.hex`）をドラッグ&ドロップすると、ファイルが転送されます。

## 起動方法

BASICモードから`MON`で機械語モニタを起動し、`G100`で起動します。<br>
または、BASICモードから`CALL &H100`でも起動できます。<br>

## ゲームのジャンル

本格サスペンスアドベンチャーゲーム<br>

## 遊び方

物語は、事件のあった花隈町からスタートします。<br>
部下のヤスが、「ボス？」と聞いてくるので、彼に命令してください。<br>

## 操作方法

### 《タイトル画面》

| キー | 役割 |
|------|------|
| `Sキー` | 音のOFF/ONの切り替え<br> ※本体に圧電ブザーが取り付けられていない場合、ONにしても音は鳴りません。 |
| `SPACEキー` | ゲームスタート |

### 《ゲーム》

入力プロンプト(`ﾎﾞｽ?`)が表示されたら、`名詞`＋`動詞`の形式で入力してEnterを押すことにより、ヤスに命令します。<br>
入力は`ローマ字`となります。<br>
（ポケコンPC-G850のローマ字入力方法に準拠しています）<br>
名詞と動詞の間は、空白を１文字入れてください。

（入力例）<br>
KIKIKOMI SHIRO → ｷｷｺﾐ ｼﾛ<br>
HANAKUMACHOU IKE → ﾊﾅｸﾏﾁｮｳ ｲｹ<br>

持っているは、「`ﾓﾁﾓﾉ`」で確認できます。<br>

以下のコマンドにより、音のON/OFFの切り替えができます。<br>
`ｻｳﾝﾄﾞ ｵﾝ`　音を出します。<br>
`ｻｳﾝﾄﾞ ｵﾌ`　音を消します。<br>

## 終了方法

本体の`BASICキー`・`TEXTキー`・`OFFキー`のいずれかを押してください。<br>

## うまく動作しない場合

このゲームでは、本体のRAMのほぼ全てを使用しています。<br>
そのため、他のプログラムは極力入れないようお願いします。<br>

## その他

* 音を出したい場合、本体へ圧電ブザーを取り付ける必要があります。<br>
圧電ブザーの取り付け方法につきましては、[作者のWebSite](http://yswgt879.blog.fc2.com/blog-entry-360.html)か、各種サイトにてご確認ください。<br>
* SHARP PC-G850/G815/E200エミュレータ g800から動作させる場合、実機ROMが無いと、入力が若干遅かったり、音割れが発生する等の問題があります。<br>
音割れが気になるようであれば、サウンドはOFFにしてください。<br>
* PC-G850Sにて、動作を確認しております。<br>
他機種（PC-G850・V・VS）での動作も可能かと存じますが、検証が取れておりません。<br>
